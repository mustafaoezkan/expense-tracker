### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mustafaoezkan/expense-tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install or yarn install
   ```
3. Run app
   ```sh
   npm start or yarn start
   ```
